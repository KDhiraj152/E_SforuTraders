# Security policy

Security matters for this project. If you find a vulnerability, please report it privately.

## Supported versions

Security fixes are applied to the `main` branch.

## How to report

Email: [k.dhiraj.srihari@gmail.com](mailto:k.dhiraj.srihari@gmail.com)

Please include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested fix (if known)

## Response targets

- Initial acknowledgement: within 3 business days
- Triage decision: within 7 business days
- Fix timeline: depends on severity and complexity

## Disclosure policy

Please do not open a public issue for unpatched vulnerabilities.

After a fix is released, coordinated public disclosure is welcome.
