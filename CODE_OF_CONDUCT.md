# Code of conduct

This project follows a simple standard: be respectful, be clear, and help move the work forward.

## Our commitment

We want this project to be welcoming for everyone, regardless of experience level, background, identity, or location.

## Expected behavior

- Use respectful and inclusive language.
- Assume good intent and ask clarifying questions before escalating.
- Focus feedback on code and behavior, not people.
- Accept constructive feedback and course-correct quickly.

## Unacceptable behavior

- Harassment, hate speech, discrimination, or intimidation.
- Personal attacks, doxxing, or threats.
- Repeated disruptive behavior after a warning.

## Enforcement

If you encounter unacceptable behavior, report it at [k.dhiraj.srihari@gmail.com](mailto:k.dhiraj.srihari@gmail.com) with links/screenshots and a short summary.

Project maintainers will review reports, respond within 5 business days, and take appropriate action, including warning, temporary restriction, or permanent ban.

## Scope

This code of conduct applies in repository discussions, issues, pull requests, and any official project communication channels.
